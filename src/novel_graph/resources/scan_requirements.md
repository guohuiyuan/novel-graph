# 扫书要求（精简版）

- 扫书目标：帮助读者快速判断是否可读、哪里爽、哪里雷。
- 结论原则：
  - 不夸大，不脑补；
  - 雷点和郁闷点分开写；
  - 信息不足要显式标注。
- 雷点采用六雷体系：绿帽、死女、送女、背叛、wrq、龟作。
- 雷点描述不能只有词条，必须落到“谁对谁做了什么事”的事件级描述。
- 防御体系参考：神防之上、神防、重甲、布甲、轻甲、低防、负防。
- 准女主判定：
  - 非wrq女性；
  - 戏份很多且颜值与女主接近；
  - 与主角存在明显感情交集，并至少满足：互相暗恋 / 明确表白 / 主角明确知道其爱意。
- 判定建议：
  - 触发雷点时，优先写劝退理由；
  - 仅触发郁闷点时，给出可抗档位；
  - 没有足够证据时，写“待复核”。
- 输出风格：
  - 先给一句话结论；
  - 用短句和列表，避免大段空话；
  - 给出“适合谁看/慎入人群”。
